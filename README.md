# python-file-organizer
# Simple Python File Organizer

A lightweight script to declutter your Downloads folder automatically.

## How it Works
The script scans a target directory, identifies files by their extensions, and moves them into categorized subfolders (Images, Documents, etc.).

## Setup
1. Clone the repo.
2. Open `organizer.py` and change the `directory` path to your desired folder.
3. Run it: `python organizer.py`

## Future Improvements
- Add a 'Watchdog' feature to run the script automatically when a new file is added.
